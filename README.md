# 1. 프로그램이란?

## 1) 프로그램
- 문제를 해결하기 위한 명령어들의 집합 / 순서도

- **프로그래밍**
  - 문제를 정의하고 명령어를 조합해 유용한 작업을 수행하는 것
  - 순서도를 작성하는 행위

- **프로그래밍 언어**
  - 컴퓨터에게 작업을 지시하고 문제를 해결하는 도구

## 2) Python
- 프로그래밍 언어 중 하나
- 읽고 쓰기 쉬운 문법
- 풍부한 커뮤니티와 생태계
- 광범위한 응용 분야
- 인공지능 기본 언어로 가장 많이 사용되는 언어
- AI 개발 도구들이 파이썬으로 제공됨

## 3) 인터프리터
- 컴퓨터는 2진수로 된 기계어로 소통함
- 파이썬 명령어를 입력하면 내부 인터프리터를 거쳐 2진수로 변환
- 변환된 명령이 컴퓨터에게 전달됨
